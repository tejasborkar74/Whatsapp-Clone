# CHATTING APPLICATION

## Features
* Authentication (Login)
* Private chat (Person to person)
* Public chat (Group chat)
* Friend Request
* Find Friends
* Can set Profile picture and Text status

## Tech Stack 
* Android studio 
* Java 
* XML
* Firebase Database (Backend)
* Firebase Authentication
